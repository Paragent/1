# Музыкальная Группа

Музыкальная группа - это коллектив музыкантов, работающий вместе для создания и исполнения музыки. Они могут записывать альбомы и проводить концерты.

## Музыкальная Группа
- **Name:** Название музыкальной группы.
- **Genre:** Музыкальный жанр, в котором работает группа.
- **Date:** Дата, когда группа была создана.
- **Country:** Страна, из которой происходит группа.
- **Num_of_members:** Количество участников группы.
- **Num_of_fans:** Количество поклонников группы.

## Участники
- **Name:** Имя участника группы.
- **Instrument:** Музыкальный инструмент, который играет участник.
- **Role:** Роль участника в группе, например, вокалист, гитарист и т.д.

## Альбом
- **Name:** Название альбома.
- **Year:** Год выпуска альбома.
- **Songs:** Список песен на альбоме.

## Концерт
- **Date:** Дата проведения концерта.
- **Place:** Место, где будет проводиться концерт.
- **Ticket_price:** Цена за билет.
- **Num_of_viewers:** Количество зрителей.

## Менеджер
- **Name:** Имя менеджера группы.
- **Contact:** Контактная информация менеджера.

## Продюсер
- **Name:** Имя продюсера.
- **Company:** Компания, которая финансирует и продвигает группу.
- **Contact:** Контактная информация продюсера.
- **Money:** Количество денег у продюсера.

## Песня
- **Name:** Название песни.
- **Duration:** Продолжительность песни в секундах.
